# 🤖 AI Resume Screening & ATS System

An AI-powered Resume Screening and ATS Score Checker built using **Python**, **Streamlit**, **Scikit-learn**, and **PDF processing**. The application analyzes resumes against job descriptions, calculates ATS match scores, identifies technical skills, provides personalized improvement suggestions, and generates downloadable PDF reports.

## ✨ Features

- 🎯 ATS Match Score Calculation
- 📄 Resume Parsing from PDF
- 🛠 Automatic Skills Detection
- ❌ Missing Skills Identification
- 💡 Resume Improvement Suggestions
- 👤 Candidate Details Extraction
- 📊 Resume Statistics Dashboard
- 📥 Downloadable PDF Report
- ⚡ User-Friendly Streamlit Interface

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- PDFPlumber
- ReportLab
- Regular Expressions (Regex)

```bash

```

## 📷 Sample Workflow

1. Enter the Job Description.
2. Upload a Resume (PDF).
3. Click **Check Match**.
4. View ATS Score, Skills, Missing Skills, and Suggestions.
5. Download the Resume Screening PDF Report.
